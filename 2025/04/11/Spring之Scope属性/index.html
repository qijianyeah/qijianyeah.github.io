Spring之Scope属性

发表于2025-04-11|更新于2025-04-12|Spring 基础知识
|浏览量:

Scope属性

  • < bean>的一个属性,控制如何实例化对象

Scope属性可取值：

  • singleton: 默认值,单例的.使用ApplicationContext启动时实例化对象
  • prototype:原型.每次调用时实例化.
  • request:每次请求时实例化对象.
  • session:每次产生session(HttpSession)时实例化
  • application:产生application对象时实例化.一般都实例化一次.
  • golbal session: 全局Golbal Session,web应用中一个新的全局HttpSessin对象.在spring-webmvc-portlet提供

这些都是 Spring 框架中 Bean 的作用域（Scope） 属性的可选值，它们决定了 Spring 容器是如何创建和管理 Bean 实例的。下面我来详细解释一下每个作用域的含义、适用场景以及注意事项：

基础类：

@Component
public class MyBean {
    private final String id = UUID.randomUUID().toString();
        public String getId() {
        return id;
        }
}

1. singleton（单例） – 默认值

  • 解释：Spring 容器中只创建 一个共享的 Bean 实例，所有请求（无论是单线程还是多线程）都返回这个实例。

  • 实例化时机：在容器启动时就实例化（如果是 ApplicationContext 启动）

  • 适用场景：无状态、线程安全的 Bean（如 Service 层）

  • 示意：

说明：无论访问几次 /singleton，得到的 ID 是一样的。

2. prototype（原型）

  • 解释：每次调用 Bean 时都会创建一个新的实例。Spring 管理创建动作，生命周期由调用者负责（不会自动销毁）

  • 实例化时机：每次调用时才会实例化

  • 适用场景：有状态的对象或非线程安全的场景

  • 示意：

    说明：每次访问 /prototype 会返回不同的 ID。

3. request（请求作用域）

  • 解释：每一个 HTTP 请求都会创建一个新的 Bean 实例，请求结束后，该 Bean 被销毁

  • 使用限制：只能用于 Web 应用，并且需要 Spring Web 环境

  • 适用场景：如 Controller 层中和用户请求强相关的对象

  • 示意（Web 项目中有效）：

    说明：每次 HTTP 请求都会生成一个新实例。

4. session（会话作用域）

  • 解释：每一个 HTTP 会话（HttpSession）拥有一个 Bean 实例

  • 使用限制：只能用于 Web 环境

  • 适用场景：需要保存用户会话状态的组件（如购物车）

  • 示意：

    说明：每个会话有自己的实例，同一个浏览器刷新页面 ID 不变，换浏览器 ID 会变。

5. application（应用作用域）

  • 解释：Bean 的生命周期与 ServletContext 相同。整个 Web 应用共享一个实例（类似 singleton，但受 Web 容器管理）

  • 适用场景：全局共享资源，且是 Web 应用

  • 示意：

    说明：整个 Web 应用范围内只有一个实例，类似 singleton，但作用域更明确是 Web 全局。

6. globalSession（全局会话作用域）

  • 解释：仅在 Portlet 应用中使用，代表一个全局的 HTTP Session（多个 Portlet 共享）。

  • 注意：在普通的 Web 应用中无效，只适用于使用 Spring Portlet MVC 的场景。

  • 示意：

    ⚠️ 注意：globalSession 只在 Portlet 环境中（如 Liferay 等门户系统）使用。Spring MVC 普通 Web 项目无法使用该作用域，会抛异常。

🌟 简单对比总结

Scope 是否 Web 环境 实例数量 生命周期
singleton 否 容器中一个 容器启动到销毁
prototype 否 每次请求新建 调用者负责管理生命周期
request 是 每个 HTTP 请求一个 请求完成后销毁
session 是 每个会话一个 会话结束后销毁
application 是 全 Web 应用共享 应用结束时销毁
globalSession 是（Portlet） 全局会话共享 Portlet 全局会话结束销毁
