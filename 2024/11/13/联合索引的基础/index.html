иҒ”еҗҲзҙўеј•еңЁ MySQL дёӯзҡ„дҪҝз”ЁеҸҜиғҪдјҡеҸ—еҲ°жҹҘиҜўжқЎд»¶зҡ„еҪұе“ҚпјҢд»ҺиҖҢеҜјиҮҙзҙўеј•еӨұж•ҲгҖӮдәҶи§ЈиҒ”еҗҲзҙўеј•еӨұж•Ҳзҡ„еңәжҷҜжңүеҠ©дәҺдјҳеҢ–ж•°жҚ®еә“жҹҘиҜўжҖ§иғҪгҖӮ

иҒ”еҗҲзҙўеј•зҡ„еҹәзЎҖ

еҒҮи®ҫжңүдёҖдёӘиЎЁ my_tableпјҢдёҠйқўжңүдёҖдёӘиҒ”еҗҲзҙўеј• idx_abcпјҢе®ғеҢ…еҗ«еҲ— a, b, cгҖӮиҜҘзҙўеј•жҢүз…§ (a, b, c) зҡ„йЎәеәҸеӯҳеӮЁгҖӮ

иҒ”еҗҲзҙўеј•еӨұж•Ҳзҡ„еёёи§ҒеңәжҷҜ

  1. еҲ—йЎәеәҸе’ҢжҹҘиҜўжқЎд»¶пјҡ

    • иҒ”еҗҲзҙўеј•еңЁдҪҝз”Ёж—¶йңҖиҰҒйҒөеҫӘзҙўеј•еҲ—зҡ„йЎәеәҸгҖӮеҰӮжһңжҹҘиҜўдёӯи·іиҝҮдәҶзҙўеј•дёӯзҡ„жҹҗдёҖеҲ—пјҢеҗҺйқўзҡ„еҲ—дёҚдјҡдҪҝз”Ёзҙўеј•гҖӮ
    • дҫӢеҰӮпјҢеҰӮжһңеҸӘжңү a е’Ң b еҲӣе»әдәҶзҙўеј•пјҢиҖҢжҹҘиҜўдёӯеҸӘдҪҝз”ЁдәҶ b еҲ—жқЎд»¶пјҢзҙўеј•е°ҶдёҚдјҡз”ҹж•ҲгҖӮ

  2. иҢғеӣҙжҹҘиҜўпјҡ

    • еңЁиҒ”еҗҲзҙўеј•дёӯпјҢдёҖж—ҰжңүдёҖеҲ—дҪҝз”ЁдәҶиҢғеӣҙжҹҘиҜўпјҲеҰӮ <, >, BETWEENпјҢLIKE зӯүпјүпјҢеҗҺйқўзҡ„еҲ—е°ҶдёҚеҶҚдҪҝз”Ёзҙўеј•гҖӮ

  3. NULL еҲӨж–ӯпјҡ

    • еҜ№дәҺ IS NULL е’Ң IS NOT NULL жқЎд»¶пјҢзҙўеј•зҡ„дҪҝз”ЁеҸҜиғҪдјҡжңүжүҖдёҚеҗҢпјҢе…·дҪ“еҸ–еҶідәҺзҙўеј•зұ»еһӢе’ҢжҹҘиҜўдјҳеҢ–еҷЁзҡ„йҖүжӢ©гҖӮ

е…·дҪ“й—®йўҳеҲҶжһҗ

еҜ№дәҺжҹҘиҜў b IS NULL е’Ң a = xx AND b = xx OR c = xxпјҢжҲ‘д»¬жқҘеҲҶжһҗжҳҜеҗҰдјҡдҪҝз”ЁиҒ”еҗҲзҙўеј• (a, b, c)гҖӮ

  1. **b IS NULL**пјҡ

    • еҜ№дәҺ b IS NULLпјҢеҰӮжһңзҙўеј•жҳҜ (a, b, c)пјҢе№¶дё”жІЎжңүеҜ№ a жҸҗдҫӣжқЎд»¶пјҢйӮЈд№ҲеңЁеӨ§еӨҡж•°жғ…еҶөдёӢпјҢиҝҷдёӘжҹҘиҜўдёҚдјҡдҪҝз”ЁеҲ°иҒ”еҗҲзҙўеј•пјҢзү№еҲ«жҳҜеңЁдј з»ҹзҡ„ B-Tree зҙўеј•дёӯгҖӮ

  2. **a = xx AND b = xx OR c = xx**пјҡ

    • еҜ№дәҺ a = xx AND b = xx OR c = xxпјҢжҹҘиҜўдјҳеҢ–еҷЁдјҡе°Ҷе…¶еҲҶдёәдёӨдёӘйғЁеҲҶжқҘеӨ„зҗҶпјҡ
      • a = xx AND b = xx
      • c = xx
    • иҝҷйҮҢжңүдёҖдёӘ OR жқЎд»¶пјҢMySQL зҡ„жҹҘиҜўдјҳеҢ–еҷЁеңЁйҒҮеҲ° OR ж—¶пјҢеҫҖеҫҖдјҡж”ҫејғдҪҝз”ЁиҒ”еҗҲзҙўеј•иҖҢж”№дёәе…ЁиЎЁжү«жҸҸпјҢзү№еҲ«жҳҜеҪ“ OR зҡ„жқЎд»¶дёҚиғҪйғҪдҪҝз”ЁеҲ°еҗҢдёҖдёӘзҙўеј•ж—¶гҖӮ
    • еҰӮжһң a = xx AND b = xx йғЁеҲҶеҸҜд»Ҙжңүж•ҲдҪҝз”ЁеҲ° (a, b, c) зҙўеј•пјҢдҪҶжҳҜ c = xx дёҚиғҪеҚ•зӢ¬дҪҝз”ЁиҝҷдёӘзҙўеј•пјҢеӣ жӯӨж•ҙдҪ“дёҠеҸҜиғҪдјҡйҖүжӢ©е…ЁиЎЁжү«жҸҸгҖӮ

з»“и®ә

  1. **b IS NULL**пјҡ

    • иҝҷдёӘжҹҘиҜўеҫҲеҸҜиғҪдёҚдјҡдҪҝз”ЁиҒ”еҗҲзҙўеј• (a, b, c)пјҢеӣ дёәжІЎжңүеҜ№зҙўеј•зҡ„еүҚеҜјеҲ— a жҸҗдҫӣжқЎд»¶гҖӮ

  2. **a = xx AND b = xx OR c = xx**пјҡ

    • з”ұдәҺеӯҳеңЁ OR жқЎд»¶пјҢе№¶дё” c = xx дёҚиғҪеҲ©з”Ё (a, b, c) зҙўеј•зҡ„еүҚеҜјеҲ—пјҢеӣ жӯӨжҹҘиҜўдјҳеҢ–еҷЁеҸҜиғҪдёҚдјҡдҪҝз”ЁиҒ”еҗҲзҙўеј•пјҢиҖҢжҳҜйҖүжӢ©е…ЁиЎЁжү«жҸҸгҖӮ

дјҳеҢ–е»әи®®

  1. йҒҝе…ҚдҪҝз”Ё OR жқЎд»¶пјҡ

    • е°ҪйҮҸйҒҝе…ҚеңЁжҹҘиҜўдёӯдҪҝз”Ё OR жқЎд»¶пјҢеҸҜд»ҘйҖҡиҝҮ UNION еҲҶејҖдёӨдёӘжҹҘиҜўжқҘдјҳеҢ–пјҢеҰӮпјҡ

  2. зҙўеј•иҰҶзӣ–пјҡ

    • и®ҫи®ЎиҒ”еҗҲзҙўеј•ж—¶пјҢиҖғиҷ‘жҹҘиҜўзҡ„е…·дҪ“дҪҝз”Ёжғ…еҶөпјҢзЎ®дҝқжңҖе·ҰеүҚзјҖеҺҹеҲҷгҖӮ

  3. EXPLAIN еҲҶжһҗпјҡ

    • дҪҝз”Ё EXPLAIN е‘Ҫд»ӨжқҘеҲҶжһҗжҹҘиҜўи®ЎеҲ’пјҢзЎ®дҝқзҙўеј•зҡ„дҪҝз”Ёжғ…еҶөгҖӮ
